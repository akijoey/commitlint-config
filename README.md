Personal [commitlint](https://github.com/conventional-changelog/commitlint) config.

## Install

`$ npm i @akijoey/commitlint-config -D`

## Usage

Add to your commitlint config:

```json
{
  "extends": "@akijoey/commitlint-config"
}
```
